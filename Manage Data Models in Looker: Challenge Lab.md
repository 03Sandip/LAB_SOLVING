

Part 1: Creating dimension and measure for calculating profit and total profit



```bash
dimension: profit {
label: "profit"
description: "sgggf"
type: number
sql: ${sale_price} - ${products.cost} ;;
value_format_name: usd
}

measure: total_profit {
label: "total_profit"
description: "sgggf sum"
type: sum
sql: ${profit} ;;
value_format_name: usd
}

```
